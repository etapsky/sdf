## [0.1.0] — 2026-03-19

### Added

- Custom editor for `.sdf` files — visual PDF preview or meta/data/schema fallback
- `SDF: Inspect File` command — meta, schema summary, data tree
- `SDF: Validate File` command — structural + schema validation
- `SDF: Open Preview` command — open with custom editor
- SDF Document tree view in Explorer — meta, schema, data layers
- Status bar — document version, type, signed status when .sdf is active
- Context menu for .sdf files in Explorer
- Configuration: `sdf.statusBar.enabled`
